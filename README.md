An AI-powered drug discovery dashboard built for **VIT Chennai MDP PROJECT**. 
This app uses **RDKit** and **Plotly** to perform Lipinski diagnostics 
and ADME mapping (BOILED-Egg) for CNS-active leads.

## Features
* **AI Structural Scouring**: Translates screenshots to SMILES.
* **Lipinski Diagnostics**: Real-time filtering (MW, LogP, HBD, HBA).
* **ADME Mapping**: Visual BOILED-Egg plot for BBB permeability.

##  Tech Stack
* Python (Streamlit)
* RDKit (Cheminformatics)
* Plotly (Visualizations)
